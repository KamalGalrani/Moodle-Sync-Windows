What is it?

Moodle Sync is a free windows software released under GPL v3.  It automatically downloads course content and forum posts for registered courses from IIT Bombay Moodle Site at regular intervals and shows notifications for new materials/posts.

Though it is being developed for IIT Bombay, it may also work on other similar moodle sites.

$ cd your_repo_root/repo_name

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

How to install?

  1. Download the appropriate zip and extract it.
  2. Run setup.exe. If there is an error try manual install.
  3. Setup Moodle Sync.
